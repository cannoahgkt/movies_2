Endpoints Explained

Return a list of all movies

  • Description: This endpoint returns a list of all movies in the database, complete with details such as Title, Director, Genre, Year, Description, ImageUrl, and Featured.
  • URL: /movies
  • Method: GET
  • Query Parameters: none
  • Request Body: none
  • Response Body: JSON object containing details for all movies in the database.
    • Example:

    {
      "title": "Treasure Planet",
      "director": "John Musker, Ron Clements",

      "featured": true
    },
    {
      "title": "Star Wars: Episode III - Revenge of the Sith",
      "director": "George Lucas",

Return data about a single user by username

  • Description: This endpoint returns the details for a single user in the database, identified by their username.
  • URL: /users/:Username
  • Method: GET
  • Query Parameters: :Username
  • Request Body: none
  • Response Body: JSON object containing details for the user.
    • Example: 
      {
  "_id": "654955e502d2f93ced6e02cc",
  "Username": "JohnDoe",
  "Password": "securepass",
  "Email": "john.doe@email.com",
  "Birthday": "1995-08-25T00:00:00.000Z",
  "FavoriteMovies": [
    "6549494c02d2f93ced6e02c3",
    "65494a9102d2f93ced6e02c4"
  ]
}

Allow new users to register

  • Description: This endpoint allows new users to register with the system by providing their details.
  • URL: /users
  • Method: POST
  • Query Parameters: none
  • Request Body: JSON object with user details: Username, Password, Email, Birthday
  • Response Body: JSON object containing details of the newly registered user.
    • Example: 
      {
  "Username": "NewUser123",
  "Password": "newpass321",
  "Email": "new.user@mail.com",
  "Birthday": "1992-11-07T00:00:00.000Z",
  "_id": "654b8b77d73c652a6f31a9f2",
  "__v": 0
}
